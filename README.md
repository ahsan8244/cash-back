A solution to reduce the excessive amount of paperwork and manual labor involved in employee reimbursements

# Install dependencies
```
npm install -g yarn
yarn global add expo-cli
```

# for mobile app
```
cd mobile 
yarn install
```

# for web app
```
cd web
yarn install
```

# to test
```
cd web
yarn start
cd ..
cd mobile
expo start
```
